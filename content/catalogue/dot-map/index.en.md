Each dot represents one data point (1 dot = 1 data item), showing the distribution of data using dots (circles). No variable is assigned to the size of the dots (maps that do assign a variable to dot size are called symbol maps). Since each dot corresponds one-to-one with individual data, it is possible to search for specific items in space.
