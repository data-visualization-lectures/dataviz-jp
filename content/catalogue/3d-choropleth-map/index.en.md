A choropleth map extended into 3D by adding height. In addition to the color used in a 2D choropleth, data variables can be assigned to the height dimension. This allows for an integrated dimension where the same data variable is mapped to both color and height, or a separated dimension where different data variables are assigned to color and height independently.
