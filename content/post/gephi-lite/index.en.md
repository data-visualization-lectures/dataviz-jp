## What is this tool?

A network diagram creation tool.
This is a browser-based implementation of Gephi, which is popular as a desktop application.


## Features

- Various layout algorithms
- Style adjustments for nodes and edges
- Filters
- Metrics (network analysis)
- Export as PNG images
- Export as GEXF files


## How to use

- Load network data
- Draw and compute as a network graph
- Save as an image or network data file

## Data formats

- GEXF
- GRAPHML
- Graphology JSON
