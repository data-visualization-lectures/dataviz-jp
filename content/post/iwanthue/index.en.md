## What is this tool?

A web tool for generating and adjusting distinguishable color palettes for data visualization. It creates highly distinguishable color sets by taking human color perception (hue, saturation, lightness, etc.) into account.

## Features

- Color palette generation...Automatically generates easy-to-distinguish color combinations based on the specified number of colors.
- Color property adjustments...Design custom palettes by specifying ranges for hue, chroma (saturation), and lightness.
- Color vision deficiency support...Capable of generating palettes that accommodate color vision diversity (color vision deficiency).
- Diverse output formats...Retrieve generated palettes as JSON, CSS, RGB/HEX lists, JavaScript code, and more.

## How to use

- 1. Set generation parameters...Specify the number of colors, hue range, and saturation/lightness constraints for the palette.
- 2. Generate the palette...Colors are listed based on your settings.
- 3. Customize and adjust...Fine-tune individual colors or overall distribution to optimize visibility and color balance.
- 4. Export and use...Export in JSON, CSS, JavaScript, and other formats for use in data visualizations.
