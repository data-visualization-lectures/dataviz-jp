## What is this tool?

JSON Crack is a web-based data visualization tool. It is designed to help you visually understand JSON data structures in your browser.

## Features


- Visualizes JSON and structured data as node-and-edge graphs or trees, with expand and collapse capabilities.


## How to use

- 1. Input data...Paste JSON or upload from a file.
- 2. An interactive node diagram or tree is rendered based on the input content.
- 3. Expand/collapse, search, zoom, and other operations as needed.



## Data formats

- JSON
